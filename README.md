### 标准格式

># 【模块名称】
>## 3.1 【功能】
>#### 3.1.1.a 【功能点，a表示前台功能，b表示后台功能,前台功能和后台功能必须一一对应】
>##### 功能描述
>就是这么屌，你服不服

>##### *关联功能*
>主要描述前置功能，不用描述相关后台功能
